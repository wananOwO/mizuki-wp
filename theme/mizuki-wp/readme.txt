== Description ==

Mizuki is a WordPress theme ported from the Astro theme Mizuki by Matsuzaka Yuki. It features:

* Material Design 3 styling with Tailwind CSS utility classes
* Light and dark mode toggle
* Configurable accent hue via Customizer
* Timeline-style archive page
* Responsive design
* Optimized asset loading

== Installation ==

1. Upload the `mizuki-wp` folder to `/wp-content/themes/`
2. Activate the theme through the 'Appearance' menu in WordPress
3. Configure the accent color under Appearance > Customize > Mizuki Options
